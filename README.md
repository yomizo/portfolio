### 当サイトは主に下記を用いました。

Vue.js, Vuetify (UI component), webpack, firebase


#### 意識した事と学んだ事

- **vue.js**の基本
- vue.js の **単一ファイルコンポーネント** の基本
- jQueryに頼らないイベントハンドリング
- よくある機能は実装する（スムーススクロール、マテリアルデザイン、インタラクティブであること）
- **firebase** でのデプロイ基本
- フォームの基礎。**バリデーション**や**メール転送**
